# Тестовое задание

## Установка зависимостей

*Требуется python версии 3.6 или новее*

```
python3 -m venv venv
source venv/bin/activate
pip3 install -r requirements.txt
```

## Запуск

```
source venv/bin/activate
python3 run.py <filename>
```

Запуск тестового файла

```
python3 run.py example.inno
```
